## About
This client-side application work with this API: [JM.API](https://github.com/yarastqt/api)

## Install & Running & Building
Installation
```sh
$ npm i --progress=false
```
Run for development
```sh
$ npm start
```
Build for production
```sh
$ npm run build
```
